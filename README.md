Mechatronics Personal Project:
My main goal for this project is to better grasp all parts of mechatronics, such as Electrical, Software, and Mechanical. 
With this first version, I didn't yet CAD the whole robot and 3D printed an open-source arm as a means of building. 

Skills I've honed and hope to learn are the following:
3D CAD - SolidWorks, OnShape, Fusion360
3D Printing - Finding correct filaments for better stability 
C++ Coding Language

Ultimately, I'll be documenting the whole process in an Engineering Notebook as a way to reason about specific parts and changes.
For a final product, I simply want to be able to grab and drop things, using a 6-degree-of-freedom robot. 

Moving forward, I'm going to take notes and review Stanford's Robotics Course and learn more about Forward and Inverse Kinematics, along with
more specialization in typical Statics and Dynamics college courses.
